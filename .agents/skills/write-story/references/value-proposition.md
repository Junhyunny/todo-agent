# Value Proposition (가치 제안)

이 파일은 `/write-story` 스킬의 선택 파일이다.
파일이 없어도 스킬이 실행되지만, 있으면 스토리 작성 시 일관된 방향성을 제공한다.

팀이 검증하고 싶은 가정(assumptions)과 제품이 제공하는 핵심 사용자 가치(user value)를 정의한다.
스킬은 이 파일을 읽어 스토리의 "so that" 부분과 검증할 가정을 제안하는 데 활용한다.

## 검증할 가정 (Assumptions to Validate)

아직 확인되지 않은 가설 목록이다.
스토리 작성 시 어떤 가정을 검증하는 스토리인지 연결하는 데 활용한다.

각 가정은 다음 형식을 따른다:
**"우리는 [사용자]가 [행동]할 것이라고 가정한다. 이것이 사실이면 [비즈니스 결과]가 나타날 것이다."**
