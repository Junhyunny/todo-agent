# 페르소나 정의

이 파일은 `/write-story` 스킬의 필수 파일이다.
파일이 없으면 스킬을 실행할 수 없다.

각 페르소나는 제품을 사용하는 실제 사람의 역할, 목표, 배경을 담는다.
팀이 공통으로 참조할 수 있도록 구체적으로 작성한다.
